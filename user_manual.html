Introduction

Welcome to the Global Reciprocal College (GRC) Student Portal! This comprehensive web-based system is designed to streamline attendance monitoring and academic management for our institution.

System Overview

The GRC Student Portal is a role-based attendance monitoring system that provides different functionalities for Administrators, Professors, and Students. The system features real-time attendance tracking, comprehensive analytics, and an intuitive user interface.

Role-Based Access

Secure access control for Admin, Professor, and Student roles

Real-time Analytics

Comprehensive attendance reports and data visualization

Responsive Design

Works seamlessly on desktop, tablet, and mobile devices

Secure Portal

Protected student data with encrypted authentication

Getting Started

Login Page

The login page is the entry point to the GRC Student Portal. It features a modern design with the college logo and key features highlighted.

Login Page Screenshot

Features college branding, login form, and feature highlights

How to Login:

  1. Open your web browser and navigate to the GRC Student Portal URL
  2. Enter your email address in the "Email Address" field
  3. Enter your password in the "Password" field
  4. Click the "Sign In" button to access your dashboard
  5. If you encounter an error, check your credentials and try again
Note: If you don't have an account, click the "Create account" link to register as a new user.

Administrator Guide

Admin Role

Admin Dashboard Overview

As an administrator, you have access to comprehensive system management tools and overview statistics.

Dashboard Statistics Include:

  • Total Students
  • Total Professors
  • Total Classes
  • Total Subjects

Admin Functions:

  • Manage Professors: Add, edit, and remove professor accounts
  • Manage Students: Add, edit, and remove student accounts
  • Manage Schedule: Create and modify class schedules
  • View Recent Activities: Monitor system activities and changes

Navigation for Admins

Use the sidebar menu to access different management sections:

  • Dashboard: Overview of system statistics
  • Manage Professors: Professor account management
  • Manage Students: Student account management
  • Manage Schedule: Class schedule management

Professor Guide

Professor Role

Professor Dashboard Overview

Professors can view attendance reports, analytics, and manage their classes and subjects.

Key Features:

  • Attendance Reports: View attendance statistics per subject
  • Analytics Charts: Pie charts for attendance distribution, bar charts for monthly trends
  • Student-wise Statistics: Individual student attendance records
  • Recent Sessions: Quick access to recent attendance data

Managing Attendance

Professors can take attendance and view detailed reports:

  1. Navigate to "Manage Class" from the sidebar
  2. Select the class and date for attendance
  3. Mark attendance for each student (Present, Absent, Late, Excused)
  4. Add remarks if necessary
  5. Save the attendance record

Professor Navigation

  • Dashboard: Attendance reports and analytics
  • Manage Subjects: Add and manage course subjects
  • Manage Class: Take attendance and manage class schedules
  • Settings: Update personal information

Student Guide

Student Role

Student Dashboard Overview

Students can view their personal attendance records and enrolled classes.

Dashboard Features:

  • Per-Subject Attendance: Attendance rate and breakdown for each enrolled subject
  • Overall Analytics: Pie chart for attendance distribution, bar chart for monthly trends
  • Recent Attendance: Table of recent attendance records
  • Enrolled Classes: List of all enrolled classes with professor and schedule information

Viewing Attendance Details

Students can view detailed attendance information:

  1. Click on "View" button next to any attendance record
  2. A modal window will open showing attendance details
  3. Review your status and any remarks from the professor
  4. Close the modal when finished

Student Navigation

  • Dashboard: Personal attendance overview
  • My Subjects: View enrolled subjects and schedules
  • Settings: Update personal information

Navigation and Common Features

Sidebar Navigation

All users have access to a sidebar menu for navigation:

  • Hamburger Menu: Click the three lines (☰) to toggle the sidebar on mobile devices
  • Active Page: The current page is highlighted in the sidebar
  • Responsive Design: Sidebar collapses automatically on smaller screens

User Dropdown Menu

Located in the top-right corner of the page:

  • Settings: Access to update personal information
  • Logout: Securely log out of the system

Common UI Elements

  • Cards: Information is displayed in card-based layouts
  • Charts: Analytics are visualized using interactive charts
  • Modals: Detailed information appears in popup windows
  • Tables: Data is presented in organized tables
  • Buttons: Actions are triggered by clearly labeled buttons

Support and Contact

Getting Help

If you encounter any issues or need assistance:

  • Check this manual: Review the relevant section for your user role
  • Contact Administrator: Reach out to the system administrator for technical support
  • Report Issues: Use the feedback system to report bugs or suggest improvements

Best Practices

  • Always log out when finished using the system
  • Keep your login credentials secure and confidential
  • Regularly check your attendance records for accuracy
  • Report any discrepancies to your professor or administrator
